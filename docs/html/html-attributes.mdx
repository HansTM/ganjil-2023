# HTML - Atribut HTML
Untuk mengubah properti atau ciri dari sebuah elemen HTML, kalian dapat menggunakan *HTML attributes* untuk memberikan informasi tambahan mengenai sebuah elemen yang ingin ditampilkan.

### Struktur Tag HTML Beratribut
Sebuah elemen HTML dapat memiliki lebih dari 1 atribut, dimana setiap atribut dipisah dengan spasi ataupun enter.

#### Contoh 1
```html
<a href="https://youtube.com" target="_blank">Link</a>
```
Anchor tag tersebut memiliki 2 atribut, yakni:
- `href` yang mengatur tujuan dari link tersebut
- `target` yang mengatur dimana link tersebut akan dibuka

#### Contoh 2
Atribut HTML dapat juga dipisahkan dengan menggunakan newline (enter) seperti pada contoh berikut:
```html
  <a
    href="https://youtube.com"
    target="_blank"
    style="color: red;"
  >
    Link
  </a>
```
Penulisan tag dengan struktur tersebut berguna ketika sebuah elemen memiliki banyak atribut dan/atau atribut yang *value*-nya sangat panjang.
